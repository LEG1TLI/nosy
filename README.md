nosy stands for Non-executing Observer for Storage Yields. This is a tool made to take untrusted usb drives and view their contents, including images, directly through your terminal using the timg ASCII image viewer. This shields your device from any of the contents of the drive that may harm your device. 

NOTES
this is NOT finished by any means. I will keep updating this to add for the feature to turn the contents of the usb drive into a web server as well. 

must have timg installed (sudo apt-get install timg)
